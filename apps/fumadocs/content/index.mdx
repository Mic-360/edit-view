# Image Editor Web App

This project is a full‑stack image editing demo using:

- TanStack Start + React for the frontend
- Hono + tRPC + Bun for the backend
- Google Gemini for AI image generation/editing
- Fumadocs for documentation
- Turborepo for monorepo tooling

The app lets you upload one image, describe an edit prompt, and receive an AI‑generated edited result you can preview and download.

## Features

- Single image upload (.jpg, .jpeg, .png, .webp), 5MB limit
- Prompt text area, 500 char limit
- Submit button disabled until valid
- Result preview + download
- Accessible forms and labels
